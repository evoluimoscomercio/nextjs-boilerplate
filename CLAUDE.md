# Regras de Escrita e Estilo

## Texto

- **Nunca separar palavras em duas linhas** (sem hifenização automática). Usar `hyphens: none; word-break: normal;` no CSS global. Não usar `word-break: break-word` pois parte palavras a meio.
- **Nunca usar travessões** (— ou –) em texto corrido. Substituir sempre por vírgula ou reestruturar a frase.
